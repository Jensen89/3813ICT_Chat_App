Phase 1 Implementation

This is Phase 1 of the chat application. The following features are implemented:

  • User authentication and registration
  • Role-based access (Super Admin, Group Admin, User)
  • Group and channel management
  • User assignment to groups
  • All data stored in localStorage

Phase 2 will include: Real-time chat, MongoDB, Socket.io, image uploads, and video chat.
